# 📝 Todo List Application

A modern desktop Todo List application built with **Python**, **Tkinter**, and **ttkbootstrap**.

## 📸 Application Preview

![Home Screen](images/home.png)

## ✨ Features

- ✅ Add Tasks
- ✏️ Edit Tasks
- 🗑 Delete Tasks
- ✔️ Mark Tasks as Completed
- 🚩 Flag Important Tasks
- 🔍 Search Tasks
- 📂 Organize Tasks into Lists
- 🍅 Pomodoro Timer
- 💾 Automatic Task Saving

## 🛠 Technologies Used

- Python
- Tkinter
- ttkbootstrap

## 🚀 How to Run

```bash
pip install -r requirements.txt
python Main.py
```
